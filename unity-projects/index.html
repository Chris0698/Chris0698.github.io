Unity Projects

In my spare time I have worked with the Unity Game Engine and done two small projects. Both projects extended the basic tutorials to include other features such as extra game features and interactivity.

My first project I extended a simple asteroid shooter game tutorial foundhere. For this game, I added an interactive game menu, another game mode where you had to exceed a goal to beat the level, and I also added special abilities to the player ship. The special ability was made with flexibility in mind and can be replaced with a different or new ability. Finally, I made the game playable on mobile devices.

Date

2018-2019

The second project I extended the survival shooter tutorial locatedhere. This time i added "boss" like enemies and multiple levels of difficulties. The highest score for the played level is also saved in JSON form and is displayed for each level in the level select.
